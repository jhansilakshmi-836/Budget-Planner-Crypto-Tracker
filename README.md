# Budget-Planner-Crypto-Tracker
It combines personal budgeting + realtime crypto tracking in a single platform.
***Budget Planner Section
User enters income or expense with name, amount, and type.
The system records the data with current timestamp.
It calculates:
Total Income
Total Expense
Final Balance
Displays values and shows a pie chart (income vs expense).
***Crypto Tracker Section
User enters crypto name (like bitcoin) and amount owned.
The system calls the CoinGecko API to get live price.
It calculates total value of holdings.
Shows a portfolio table + pie chart of crypto value distribution.
